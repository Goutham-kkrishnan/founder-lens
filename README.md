# FounderLens

FounderLens is a guided decision-making system for early-stage startup founders.

## Problem
Founders make critical decisions without structured thinking, context, or access to relevant case studies.

## Solution
FounderLens provides:
- Guided decision workflows
- AI-generated structured reasoning
- Decision journaling
- Case-based insights

## MVP Features
- Decision input flow
- AI-generated analysis
- Save decisions
- Basic case suggestions

## Tech Stack
- Next.js (Frontend)
- FastAPI (Backend)
- PostgreSQL (Database)

## Vision
To build an AI-powered decision intelligence system for founders.
